# Herní nápověda – přehled složky

Tahle složka sdružuje všechny podklady pro nápovědu do hry Agent USA.

- `game_system_overview.md` – hlavní dokument, ve kterém popíšeš celkový princip hry (dodáš ho Ty).
- `buildings.md` – připravený prostor pro popisy jednotlivých budov / lokalit.
- `materials.md` – prostor pro vysvětlení materiálů, zdrojů a jejich získávání.
- `tools.md` – prostor pro návody k nástrojům/dílenským prvkům.
- `regions.md` – prostor pro poznámky k regionům a cestování.

Všechna `.md` jsou zatím prázdná nebo jen s nástřelem struktury – můžeš je naplnit vlastním obsahem.
